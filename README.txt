Prerequisites.

Recomended Hardware

Minimum System Requirements[2]
Operating system	Windows 7,8 or 8.1, 10
Processor	2.5+ Ghz Processor
Ram	4GB
GPU	AMD Radeon HD 6950 or above
Or
NVIDIA GeForce GTX 560 or above

These are required to run the project.
Please follow the instructions closly.

Oculus Runtime version : 0.8 Beta
Install Link : https://developer.oculus.com/downloads/pc/0.8.0.0-beta/Oculus_Runtime_for_Windows/
Follow the instructions on the website.

Graphics Drivers
If you have an NVIDIA GPU, make sure to upgrade to the 358.70 driver or later. To get the driver, 
go to https://developer.nvidia.com/gameworks-vr-driver-support.

or

If you have an AMD GPU, we recommend the Catalyst 15.10 Beta or later. To get the driver,
go to http://support.amd.com/en-us/kb-articles/Pages/latest-catalyst-windows-beta.aspx .

#---------------Section Three---------------#
Wanting to view Code.

You will need to install the following software

Visual Studio 2015
https://www.visualstudio.com/post-download-vs?sku=community&clcid=0x409

Make sure to select C++ when installing Visual Studio as its not an option as of 2015.

Unreal engine Launcher

Follow the following guide
https://docs.unrealengine.com/latest/INT/GettingStarted/Installation/index.html

Make sure in the launcher to install UE4 10.4

Open the MastersProject folder
Right click the MastersProject.uproject file,
select generate visual studio project files.

Open the SLN file in visual studio you can now view the C++ code

By building and running the project the editor will appear
Here you can see blueprints and run the game.
